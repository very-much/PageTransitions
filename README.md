##简介
PageTransition是一款实现了手机浏览器中多种页面过渡效果的插件。
##如何使用？
###html
|默认类名	|作用	|
|-----------:|:----	|
|pt-main	|页面过渡容器	|
|pt-page	|页面过渡单页	|
|pt-page-current	|当前页	|

·
